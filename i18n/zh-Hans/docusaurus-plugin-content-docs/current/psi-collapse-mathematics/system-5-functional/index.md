# 系统5 – Ψhē 函数系统

*函数作为ψ坍缩算子*

## 函数原理

本系统的核心是一个深刻的洞察：

**每个函数 f: A → B 都是从叠加态到本征态的坍缩路径**

当我们写 f(x) = y 时，我们描述的不是查找表而是坍缩事件——输入态 x 通过函数算子 f 共振，坍缩为输出态 y。整个函数分析的装置都从这个量子视角涌现。
